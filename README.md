# Mini Pauta Escolar

Projeto para fazer o cálculo o rápido de media de estudantes e gerar a mini pauta.

# Funcionalidades

- Registro de Alunos
- Registro de Disciplina
- Lançamento de Nota
- Calculo de Média e Resultado Final

# Stack

- React.JS
- React Hooks
- React-Select
- LocalStorage (API)
- StyledComponents
- ReactStrap
